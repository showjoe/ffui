Modal

Use the Modal component to add dialogs to your site for lightboxes, user notifications, or completely custom content..

# Inline

Below is a static modal example (meaning its position and display have been overridden). Included are the modal header, modal body (required for padding), and modal footer (optional). We ask that you include modal headers with dismiss actions whenever possible, or provide another explicit dismiss action.

Modal Title
Some example text in the body
hide code 
<modal inline show title="Modal Title">
	Some example text in the body
</modal>

# Triggered by a click

Toggle a working modal demo by clicking the button below. It will slide down and fade in from the top of the page.

hide code 
<div>
	<btn @click.native="modal1.show = true">Show</btn>
	<modal title="Modal Title" :show="modal1.show" @close="modal1.show = false" @save="modal1.saved = true; modal1.show = false" >
		Some body to my modal
	</modal>
</div>

# backdrop-inactive

When backdrop-inactive prop is set, the modal will not close when clicking outside it. Click the button below to try it.

hide code 
<div>
	<btn @click.native="modal2.show = true">Show with inactive background</btn>
	<modal title="Modal Title" :show="modal2.show" backdrop-inactive @close="modal2.show = false">
		Some body to my modal
	</modal>
</div>

# backdrop-hidden

When backdrop-hidden prop is set, the modal will not display the darkened backdrop.

hide code 
<div>
	<btn @click.native="modal3.show = true">Show with hidden background</btn>
	<modal title="Modal Title" backdrop-hidden :show="modal3.show" @close="modal3.show = false">
		Some body to my modal
	</modal>
</div>

# Scrolling long content

When modals become too long for the user's viewport or device, they scroll independent of the page itself. Try the demo below to see what we mean.

hide code 
<div>
	<btn @click.native="modal4.show = true">Show Modal</btn>
	<modal :show="modal4.show" @close="modal4.show = false">
		<p v-dummy:350></p>
	</modal>
</div>

You can add a scrollable prop to the modal and the dialog window will scroll.

hide code 
<div>
	<btn @click.native="modal5.show = true">Show Modal</btn>
	<modal scrollable :show="modal5.show" @close="modal5.show = false">
		<form-group>
			<textbox />
		</form-group>
		<p v-dummy:350></p>
	</modal>
</div>

# Tooltips and popovers

Tooltips and popovers can be placed within modals as needed. When modals are closed, any tooltips and popovers within are also automatically dismissed.

hide code 
<div>
	<btn @click.native="modal7.show = true">Show</btn>
	<modal title="Modal Title" :show="modal7.show" centered @close="modal7.show = false" @save="modal7.saved = true; modal7.show = false">
		<alert type="info" class="d-flex">
		Here is a popover displayed when you click the button:
		<btn class="ml-auto" v-popover="{title:'Popover title',body:'And heres some amazing content. Its very engaging. Right?'}">Click to toggle popover</btn>
		</alert>
		Here is some text inside <b v-tooltip="{text:'And heres some amazing content. Its very engaging. Right?'}">a tooltip</b>
	</modal>
</div>

# no-blur, no-fade, no-slide

You can disable elements of the default style.

  • no-blur will disable the blurring of the background.
  • no-fade will disable the fading of the background and modal.
  • no-slide will disable the sliding of the modal
hide code 
<div>
	<btn @click.native="modal8.show = true">Show</btn>
	<modal title="Modal Title" no-blur no-fade no-slide :show="modal8.show" centered @close="modal8.show = false" @save="modal8.saved = true; modal8.show = false">
		Some body to my modal
	</modal>
</div>

# target

position the modal like a popover (using popper.js) by adding a target prop. The target prop should resolve to an html element or query string. You can also add an optional arrow prop

hide code 
<div>
	<div class="d-flex">
		<btn id="modalTarget" @click.native="modal16.show = true" class="m-auto">Show Modal with this button as it's target</btn>
	</div>
	<modal backdrop-hidden no-blur arrow allow-background-scroll :show="modal16.show" target="#modalTarget" title="Modal Title" @close="modal16.show = false">
		Some body to my modal
	</modal>
</div>

		Some body to my modal
	</modal>
